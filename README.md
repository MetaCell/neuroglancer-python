# neuroglancer-python

## Installation

```bash
git clone https://github.com/MetaCell/neuroglancer-python
cd neuroglancer-python
pip install -e .

git clone -b develop https://github.com/MetaCell/neuroglancer/tree/develop
cd neuroglancer
npm i
npm run build-python
pip install .
```
